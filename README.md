# HD2SD

Easily convert @2x (HD) images to SD.

## Usage
- Download the latest executable from the [releases](https://github.com/SnowzNZ/HD2SD/releases/latest).
- Run `HD2SD.exe`.

#### Manual Installation
```sh
git clone https://github.com/SnowzNZ/HD2SD
cd HD2SD
pip install -r requirements.txt

py main.py
```
